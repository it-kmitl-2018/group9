# Group 9: E-Tax Invoice Web Application
Use for create a electronic tax invoice in EDTA format

## Prerequisites
- Spring Boot 1.5.10
- Java SDK (1.8 or upper version)
- IntelliJ IDEA

## Built With
- Spring Boot Framework
- Java
- Gradle

## Installation
1. Clone this repository
2. Build & Run on your IDE
